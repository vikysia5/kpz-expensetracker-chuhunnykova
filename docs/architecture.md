# Архітектура проєкту Expense Tracker

## Опис домену
Система призначена для обліку особистих та спільних витрат користувачів, а також для аналізу фінансових даних.

### Яку проблему вирішує система?
Користувачам складно контролювати свої витрати та аналізувати фінансові звички. Система дозволяє зручно зберігати витрати, переглядати статистику та керувати бюджетом.

### Користувачі системи
- Гість — переглядає інформацію про застосунок
- Користувач — додає та аналізує витрати
- Спільний користувач — працює зі спільним бюджетом

### Основні операції
- Реєстрація та авторизація
- Додавання витрат
- Редагування та видалення витрат
- Перегляд статистики
- Розподіл витрат по категоріях
- Спільний доступ до витрат
- Експорт звітів

### Зовнішні системи
- База даних (MongoDB/PostgreSQL)
- Telegram API (для інтеграції)

## Модуль реєстрації користувача
Відповідає за перегляд вступного екрану та створення нового акаунту з нікнеймом і паролем.

## Модуль витрат
Відповідає за додавання, збереження та обробку витрат користувача.
