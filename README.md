# Expense Tracker

## Опис
Expense Tracker — веб-застосунок для обліку особистих витрат. Користувач може додавати витрати, переглядати статистику, аналізувати категорії витрат та отримувати звіти. Застосунок допомагає ефективно контролювати бюджет.

## Технології
- Мова програмування: Node.js 20
- Фреймворк: Express.js
- База даних: PostgreSQL
- ORM: Tortoise ORM
- ORM: SQLAlchemy

## Ролі користувачів
- Гість (Guest): неавторизований користувач, може переглядати лише загальну інформацію про застосунок, але не має доступу до власних витрат або до запису нових витрат.

- Користувач (User): зареєстрований користувач може додавати витрати, вибирати категорії, залишати опис, переглядати власні звіти та статистику.

- Спільний користувач (Shared User): зареєстрований користувач, з яким розділяється доступ до одного акаунта. Може додавати та переглядати витрати в цьому спільному обліковому записі, але не має доступу до налаштувань інших акаунтів.
